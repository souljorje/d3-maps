# docs/examples Guide

Guide for [packages/docs/examples](./).

## Scope

- Overview page: [packages/docs/examples/index.md](./index.md)
- Generated example wrappers: [packages/docs/examples/basic.md](./basic.md)

## Source of truth

- Example implementations live in `packages/docs/.vitepress/examples/*.vue`
- Wrapper pages are generated by `pnpm docs:gen:examples` via [scripts/docs-gen-examples.mjs](./../../../scripts/docs-gen-examples.mjs)

## Conventions

- Do not manually edit generated wrapper files like [packages/docs/examples/basic.md](./basic.md)
- Keep stable demo ids as `<file-slug>` and render via `<Demo component-name="...">`.
- Generated wrappers pass source snippets through Demo slots using native VitePress `::: code-group`.
