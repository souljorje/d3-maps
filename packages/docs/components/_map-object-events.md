== Vue

Emits:

- `mouseenter`
- `mouseleave`
- `mousedown`
- `mouseup`
- `focus`
- `blur`

== React

Callbacks:

- `onMouseEnter`
- `onMouseLeave`
- `onMouseDown`
- `onMouseUp`
- `onClick`
- `onFocus`
- `onBlur`

`onClick` uses the same interaction state transition as mouse up
