# d3-maps docs

Use this site for d3-maps package docs and examples.

Start with `/llms.txt` for a compact index.
Use `/llms-full.txt` for a broader docs dump.
Use page markdown routes like `/guide/architecture.md` or `/components/map-base.md` when you need page content without HTML.

Focus on:

- `/guide/` for setup and concepts
- `/components/` for component behavior
- `/helpers/` for hooks and helpers
- `/examples/` for working demos
- `/api/` for generated API docs
